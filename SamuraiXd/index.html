rules of the game

  • this game is a two player game
  • one player must kill the other before time runs out
  • player 1 controls : 'a' => left, 'd' => right 'w' => jump, 'x'=> attack
  • player 2 controls : 'keyleft' => left, 'keyright' => right 'keyup' => jump, 'keydown'=> attack
  • enjoy the game
